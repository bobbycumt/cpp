### 📘 **一、基础型（青少年等级考试常考）**

#### **1. 求整数每一位的数字之和**

例如：输入 1234 → 输出 1+2+3+4 = 10

练习题：

* 输入一个正整数，输出其各位数字的和。
* 输入 N 个数，输出每个数的数位和。
```cpp
/*---------------- 1. 各位数字之和 ----------------*/
// sumDigits(12345) = 1+2+3+4+5 = 15
int sumDigits(long long n) {
    int sum = 0;
    if (n == 0) return 0;
    while (n > 0) {
        sum += n % 10;
        n /= 10;
    }
    return sum;
}
```

#### **2. 求整数的偶数位/奇数位数字之和**

练习题：

* 输入一个正整数，求所有“偶数位上的数字之和”。
* 输入一个正整数，求所有“奇数位上的数字之和”。

---

#### **3. 求整数中某个数字出现了几次**

例如：输入 5573550，统计数字 5 出现几次。

练习题：

* 输入一个整数，统计 3 出现了几次。
* 输入若干整数，统计数字 9 出现次数最多的一个。
```cpp
/*---------------- 2. 统计某个数字出现次数 ----------------*/
// countDigit(105550, 5) = 3
int countDigit(long long n, int x) {
    int cnt = 0;
    if (n == 0 && x == 0) return 1;  // 特判 0
    while (n > 0) {
        if (n % 10 == x) cnt++;
        n /= 10;
    }
    return cnt;
}
```

#### **4. 逆序输出数字**

例如：输入 123 → 输出 321

练习题：

* 将输入的整数按位倒序打印。
* 判断是否为回文数（利用逆序）。
```cpp
/*---------------- 3. 数字反转（逆序） ----------------*/
// reverseNumber(12345) = 54321
long long reverseNumber(long long n) {
    long long rev = 0;
    while (n > 0) {
        rev = rev * 10 + (n % 10);
        n /= 10;
    }
    return rev;
}
```

#### **6. 找出数位中最大的数字 / 最小的数字**

题目示例：

* 输入一个整数，输出其中最大的数字。

类似题：

* 同时求最大位与最小位。
* 判断数位最大的是否为偶数。
```cpp
/*---------------- 5. 最大位/最小位数字 ----------------*/
// maxDigit(58042) = 8
// minDigit(58042) = 0
int maxDigit(long long n) {
    int mx = 0;
    if (n == 0) return 0;
    while (n > 0) {
        mx = max(mx, (int)(n % 10));
        n /= 10;
    }
    return mx;
}

int minDigit(long long n) {
    int mn = 9;
    if (n == 0) return 0;
    while (n > 0) {
        mn = min(mn, (int)(n % 10));
        n /= 10;
    }
    return mn;
}
```

#### **13. “幸运数”类题（非常常见）**

例如：左右两半的数位和是否相等。

题目示例：

* 判断一个 6 位数是否为“幸运票号”（前三位之和 = 后三位之和）。
* 输入 n，统计从 000000 到 n 的所有幸运号数量（基础版）。
```cpp
/*---------------- 11. “幸运数”：左右两半数位和相等 ----------------*/
// 例如 6 位数 abcdef，若 a+b+c == d+e+f 则为幸运
bool isLucky6(long long n) {
    // 假定 n 是 0 ~ 999999 之间
    int a = n / 100000 % 10;
    int b = n / 10000 % 10;
    int c = n / 1000 % 10;
    int d = n / 100 % 10;
    int e = n / 10 % 10;
    int f = n % 10;
    return (a + b + c) == (d + e + f);
}
```
