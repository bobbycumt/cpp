在 C++ 教学中，字符串的处理是学生从“纯数值运算”转向“实际应用开发”的标志。C++ 提供了两种字符串：传统的 **C 风格字符串**（`char[]`）和更现代、更安全的 **STL `std::string`**。教学应以 `std::string` 为核心。

以下是针对字符串四大核心功能的教学要点：

---

## 一、 字符串基础与大小写转换

### 1. 核心概念

- **本质**：`std::string` 是一个动态的字符序列，支持下标操作 `s[i]`。
    
- **输入陷阱**：`cin >> s` 遇到空格会停止。若要读入整行（含空格），必须使用 `getline(cin, s)`。
    

### 2. 大小写转换

C++ 标准库中没有直接的 `s.toUpper()` 函数，通常有两种教法：

- **底层逻辑法**（利用 ASCII 码）：
    
    - `'A'` 是 65，`'a'` 是 97。
        
    - 大写转小写：`s[i] += 32`；小写转大写：`s[i] -= 32`。
        
- （选讲）标准库函数法（`<algorithm>`）：
    
    使用 `transform` 配合` toupper / tolower`。
    
    ```CPP
    transform(s.begin(), s.end(), s.begin(), ::toupper);
    ```
    
### 3. 字符串比大小
- 按照顺序比较字符大小，比如`"ABC">"AACDE"`

- 完全相同才为相等

---

## 二、 搜索

### 1. 核心函数：`s.find()`

这是最常用的成员函数。

- **返回值**：如果找到，返回第一个匹配字符的**下标**；如果找不到，返回常量 `-1`
- **教学要点**：
    
    - 一定要强调如何检查“未找到”的情况：`if (s.find("abc") != -1)`。
        
    - **反向搜索**：`s.rfind()` 从右向左找。
        
    - **指定起点**：`s.find("target", pos)` 从位置 `pos` 开始往后找，常用于循环搜索所有匹配项。
        

---

## 三、 提取与分割

### 1. 提取子串：`s.substr()`

- **语法**：`s.substr(pos, len)`。
    
- **易错点**：第二个参数是**长度（`len`）**，而不是结束下标。
    

### 2. 字符串分割

C++ 没有像 Python 那样的 `s.split()`，教学中通常引导学生通过“搜索 + 提取”手动实现：

- **逻辑推导**：
    
    1. 用 `find()` 找到分隔符（如逗号）的位置。
        
    2. 用 `substr()` 截取分隔符之前的字符串。
        
    3. 更新搜索起点，循环往复。
        
- （选讲）**进阶工具**：介绍 `stringstream`（需包含 `<sstream>`），它可以像 `cin` 一样通过 `>>` 自动分割空格分隔的字符串。
    

---

## 四、 替换与插入

### 1. 替换：`s.replace()`

- **用法**：`s.replace(pos, len, "new_str")`。
    
- **注意**：这会改变原字符串。如果长度不等，会自动调整后续字符的位置。
    

### 2. 批量替换

如果要将所有的 "A" 换成 "B"，需要配合 `while` 循环和 `find` 函数不断更新位置。

### 3. 插入与删除：

- **插入**：`s.insert(pos, "str")`。
    
- **删除**：`s.erase(pos, len)`。

## 六、 `reverse函数`

- **头文件**：`#include <algorithm>`
    
- **函数原型**：`reverse(first, last)`
    
    - `first`：指向起始位置的迭代器。
        
    - `last`：指向终止位置**之后**的迭代器（即左闭右开区间 `[first, last)`）。
        
### A、 不同容器用法
### 1. 处理 `std::string`

这是最常见的使用场景，用于翻转单词或检查回文。


```CPP
string s = "Hello";
reverse(s.begin(), s.end()); // s 变为 "olleH"
```

#### 2. 处理 一维数组

对于普通数组，迭代器就是指针。


```CPP
int arr[] = {1, 2, 3, 4, 5};
int n = 5;
reverse(arr, arr + n); // arr 变为 {5, 4, 3, 2, 1}
```

#### 3. 处理 部分区间

你可以只翻转字符串或数组的一部分。



```CPP
string s = "ABCDE";
reverse(s.begin() + 1, s.begin() + 4); // 只翻转 BCD，结果为 "ADCBE"
```

---

### B、 核心逻辑与底层原理

在教学中，可以引导学生思考：**如果不调用这个函数，我们该如何手动实现它？**

- **双指针法**：设置一个头指针 `L` 和一个尾指针 `R`。
    
- **交换逻辑**：当 `L < R` 时，交换两个指针指向的元素，然后 `L++`, `R--`。
    
- **复杂度**：时间复杂度为 $O(n)$，空间复杂度为 $O(1)$。
    

---

### C、 经典应用场景

#### 1. 回文判断（最简单的解法）

将字符串拷贝一份，翻转后与原串比较。



```CPP
string s = "level";
string temp = s;
reverse(temp.begin(), temp.end());
if (s == temp) cout << "是回文";
```

#### 2. 翻转句子中的单词

**题目**：输入 `"I love C++"`，输出 `"C++ love I"`。

- **步骤 1**：翻转整个字符串 $\to$ `"++C evol I"`。
    
- 步骤 2：再次翻转每个独立的单词 $\to$ "C++ love I"。
    
    (这是一个经典的算法面试题，通过两次翻转巧妙解决。)
    

#### 3. 进制转换输出

在手动实现十进制转二进制（短除取余法）时，得到的余数是逆序的。

- **做法**：先用 `string` 记录余数，最后执行一次 `reverse` 即可得到正确顺序。
    

---

### ⚠️ 教学避坑指南

1. **左闭右开**：再次强调 `last` 指向的是最后一个元素的**下一个位置**。
    
2. **不返回值**：很多学生会写出 `s = reverse(s.begin(), s.end());`，这是错误的，因为 `reverse` 的返回值是 `void`。
    
3. **迭代器失效**：虽然 `reverse` 不会改变容器大小，但在某些复杂容器操作中，仍需注意对迭代器的理解。
    

## 🎯 课堂练习题目：

1. **敏感词过滤**：给定一段话，将其中所有的“坏词”搜索出来并用 `***` 替换。
    
2. **网址解析**：输入一个 URL（如 `https://www.code.com/index`），要求分割出协议、域名和路径。
    
3. **回文判断**：将字符串忽略大小写后，判断其正读和反读是否一致（结合 `reverse` 函数）。
